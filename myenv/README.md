# ClimaBrasil Projeções

Projeto que analisa diversas APIs de informações meteorológicas sobre o Brasil para realizar projeções das mudanças climáticas nos próximos meses, com foco nas altas temperaturas.


## Requisitos

- Python 3
- pandas
- numpy
- requests

## Instalação

1. Certifique-se de ter o Python 3 instalado. Você pode baixá-lo em [python.org](https://www.python.org/).
2. Crie e ative seu ambiente virtual.
3. Instale as bibliotecas necessárias executando o seguinte comando:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

1. Certifique-se de ter todas as dependências instaladas.
2. Execute o script principal `main.py`:
   ```bash
   python main.py
   ```

## Estrutura do Projeto

```
/
|-- main.py
|-- requirements.txt
|-- myenv/ 
|-- ...
```
